== Description ==

Display a meta box when viewing a media item in the admin that display all generated images sizes.

== Installation ==

1. Upload the `image-sizes-panel` plugin folder to your WordPress site in the `/wp-content/plugins` folder or install via the WordPress admin.
2. Activate it from the Wordpress plugin admin screen.
3. A new meta box should displayed on media pages in the admin.
